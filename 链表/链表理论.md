# 链表理论

## 一. 链表的种类：单链表，双链表，循环链表 

![dd](../图片/链表的类型.jfif)

### 1. 单链表

一个指针域，一个数据域构成一个链表节点

### 2. 双链表

两个指针域，一个数据域构成一个链表节点

### 3. 循环链表

最后的节点不指向空，而是指向第一个节点

### 4. 链表的查找、增加、删除

分别是 O(n)，O(1)，O(1)；增加节点只需要先把新节点指向下一个节点，再把前一个节点指向新节点。顺序不可反过来。

删除则只需前一个结点指向下下个结点。没有自动垃圾回收机制的语言，需要手动被跳过的结点释放内存。

### 5. 头结点和头指针和首元结点

空的头结点可以有也可以没有，没有的话，第一个结点就是首元结点，头指针指向它；有的话，第一个结点就是头节点，头指针指向它。

## 二. 链表节点的实现

一个指针域，一个数据域构成一个链表节点

```Java
Class ListNode {
    int data;
    ListNode next;
    ListNode() {

    }
    ListNode(int val) {
        this.data = val;
    }
    ListNode(int val, ListNode listNode){
        this.data = val;
        this.next = listNode;
    }
}
```
