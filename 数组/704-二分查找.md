# 704-二分查找

## 一. 题目描述

给了有序且不重复的数组，用二分法找出目标元素的下标。

## 二. 题解

### 1. 找到的时候，应该放回 mid，而不是 l。

``` Java
class Solution {
    public int search(int[] nums, int target) {
        int len = nums.length;
        int l = 0;
        int r = len - 1;
        int mid;
        while (l != r) {
            mid = (l + r) / 2;
            if (nums[mid] < target) {
                l = mid + 1;
            } else {
                r = mid - 1;
            }
        }
        return l;
    }
}

```

### 8. 利用数组已经排序，来处理不存在的情况。


``` Java
class Solution {
    public int search(int[] nums, int target) {
        int len = nums.length;
        if (target < nums[0] || target > nums[len - 1]) {
            return -1;
        }
        int left = 0;
        int right = len - 1;
        while (left <= right) {
            int mid = (left + right) / 2;
            if (target == nums[mid]) {
                return mid;
            } else if (target > nums[mid]) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        return -1;
    }
}

``` 

### 9. 除了左闭右闭，还可以左闭右开，这样因为 right 超出数组，不用考虑边界情况。
``` Java
class Solution {
    public int search(int[] nums, int target) {
        int len = nums.length;
        int left = 0;
        int right = len;
        while (left < right) {
            int mid = (left + right) / 2;
            if (target == nums[mid]) {
                return mid;
            } else if (target > nums[mid]) {
                left = mid + 1;
            } else {
                right = mid;
            }
        }
        return -1;
    }
}
``` 
